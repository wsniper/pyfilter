# pyfilter
用python实现的 [字段过滤/别名替换/默认值赋值/有效性检测] 典型应用是网站post数据过滤检测。
# 简介
功能很简单，看代码即可使用。                                                                                                                              
1. 检测必选字段。若必选字段在data中不全则返回错误信息                                                                                      
2. 字段别名替换。将别名替换为数据定义名称                                                                                                
3. 默认值赋值。 若字段值为空则赋默认值                                                                                                     4. 字段有效性检测。使用正则表达式检测
